Pfcfpy

1. 開始

1.1 載入統一API

    import pfcf
    from pfcf.pfcf import *

1.2 宣告統一API

    api = Pfcf()

1.3 註冊錯誤事件

    def on_error(err):
        print(err)
    api.on_error =on_error

1.4 系統登入

透過login 傳入參數

    loginresponse = api.login(
    "登入url", "帳號", "密碼", '憑證檔名', '憑證密碼')
