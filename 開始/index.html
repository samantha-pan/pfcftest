1. 開始

1.1 載入統一API 

    import pfcf
    from pfcf.pfcf import *

1.2 宣告統一API 

    api = Pfcf()

1.3 註冊錯誤事件 

    api.on_error = lambda error: print(f'api錯誤 {error}')

1.4 系統登入 

    loginresponse = api.login(
