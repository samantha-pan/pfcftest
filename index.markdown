## 1. 事先準備
### 1.1 儲存憑證
 - 取得您的憑證檔案和密碼
 - 將憑證檔案放到程式執行目錄下
 - 您的目錄會是

```  
 ├── 您的程式.py
 └── XXXXXXXXXX.pfx
```
### 1.2 下載pfcf套件

| 載點 |
| ------ |
| [windows](http://www.pfcf.com.tw/attachment/PFCF_API.zip) |     
| [linux](http://www.pfcf.com.tw/attachment/PFCF_API.zip) | 
| [mac](http://www.pfcf.com.tw/attachment/PFCF_API.zip) |     


### 1.3 複製pfcf套件
 
 - 複製[pfcf.whl]到程式執行目錄下   - 
 - 您的目錄會是

```  
 ├── 您的程式.py
 ├── pfcf.whl
 └── XXXXXXXXXX.pfx
```

### 1.4 安裝pfcf套件
``` 
    pip install pfcf.whl
```
