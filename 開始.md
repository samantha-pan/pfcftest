## 1. 開始
### 1.1 載入統一API
``` python
    import pfcf
    from pfcf.pfcf import *
```
### 1.2 宣告統一API
``` python
    api = Pfcf()
```
### 1.3 註冊錯誤事件
``` python
    api.on_error = lambda error: print(f'api錯誤 {error}')
```

### 1.4 系統登入
``` python
    loginresponse = api.login(
    "登入url", "帳號", "密碼", '憑證檔名', '憑證密碼')
```
